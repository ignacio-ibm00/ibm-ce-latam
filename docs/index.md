## Cómo usar este portal

- **Sellers / Arquitectos**: en cada asset encontrás el caso de uso, el one-pager ejecutivo y el diagrama de arquitectura de alto nivel.
- **AI Engineers**: al final de cada página hay una sección técnica colapsable con stack, instalación y guía de replicación.
- **Contribuir**: leé el [CONTRIBUTING.md](https://github.com/ibm-ce-latam/ce-latam-ai-portfolio/blob/main/CONTRIBUTING.md) del repositorio.
